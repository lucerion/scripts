# send_to_telegram

Sends files and messages to the [Telegram](https://telegram.org)


## Configuration

`$TOKEN` - bot token (required)

`$USER_ID` - user id (optional)


## Usage

Send message (press `Ctrl-D` to finish input)

`send_to_telegram`

Send file(s)

`send_to_telegram [PATH_TO_FILE...]`
