# send_to_telegram

Send files and messages to [Telegram](https://telegram.org)


## Configuration

`$TOKEN` - bot token (required)

`$USER_ID` - user id (optional)


## Usage

Send message (press `Ctrl-D` to send finish input)

`send_to_telegram`

Send file(s)

`send_to_telegram /path/to/file/ path/to/second/file`
